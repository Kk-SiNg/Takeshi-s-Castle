## Takeshi's Castle is one of the event of UTKARSH, the intra collage Tech fest of IIT Mandi for 1st year students. Takeshi's castle is inspired from orignal Takeshi's Castle conducted in Japan consisting of many hurdles. In our version of Takesshi's Castle We had to make a bot without using any metal screw or nut that consists of 4 different add on parts for Tug Of Wars, Baloon Pop, Robo Wrestling, Robo Soccer. For detailed info refer Rule Book.

## Please pay care to use platformio for reciever code modification or uploading into esp8266

## Hardware used:
1. ESP8266 MCU
2. BO brushed 300rpm dc motors
3. L298N motor driver
4. 2 units of 3.7V lipo battery + battery holder
5. breadboard + jumpercables

## We used one of the fastest way to controll bots motion via esp8266's builtin wifi which is UDP, with least possible latency.
